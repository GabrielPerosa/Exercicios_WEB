Ciência de Dados

Estatística

A estatística é a disciplina que utiliza métodos específicos para coletar, apresentar e interpretar conjuntos de dados, sejam estes numéricos ou de outra natureza. Seu objetivo é fornecer informações detalhadas sobre os dados analisados, possibilitando uma melhor compreensão dos fenômenos representados por esses dados. A estatística descritiva, que é a primeira fase da análise de dados, visa descrever e sumarizar os dados observados.

Estrutura de Dados

Estrutura de dados é uma maneira especifica de organizar e armazenar os dados para que sua manipulação possa ser usada de forma mais agil e eficiente.

Amostragem

A amostragem é um processo utilizado em estatística para selecionar uma parte representativa de uma população maior, com o objetivo de fazer inferências sobre essa população sem precisar examinar todos os seus elementos. Esse método é especialmente útil quando é impraticável ou impossível estudar a população inteira devido ao seu tamanho, custo ou tempo necessário.

Escalas de medição

As escalas de medição são classificações que indicam a natureza das variáveis e determinam as operações estatísticas que podem ser aplicadas a elas.

Nominal

Esta escala categoriza os dados em grupos ou categorias distintas que não possuem uma ordem ou hierarquia intrínseca.

Ordinal

Ordena os dados em uma sequência ou hierarquia, mas as diferenças entre os valores não são necessariamente uniformes ou mensuráveis.

Intervalar

Esta escala ordena os dados e a diferença entre os valores é consistente e mensurável, mas não possui um ponto zero verdadeiro.

Razão

Esta escala possui todas as características da escala intervalar, mas também possui um ponto zero verdadeiro, o que permite a comparação de proporções.

Tendências centrais

As tendências centrais são medidas estatísticas que identificam o valor central ou típico em um conjunto de dados. Elas são fundamentais para descrever a localização de uma distribuição de dados

Moda

É o valor que ocorre com mais frequência em um conjunto de dados.

Mediana

É o valor que separa a metade superior e a metade inferior dos dados, e pega o valor do meio.

Média

É a soma de todos os valores dos dados dividida pelo número de valores.

Média aritmética ponderada

Em vez de cada valor contribuir igualmente para a média, alguns valores têm mais influência do que outros, de acordo com seus pesos.

Percentil

O percentil é uma medida estatística que indica a posição relativa de um valor dentro de um conjunto de dados.

Quartil

Os quartis são medidas estatísticas que dividem um conjunto de dados em quatro partes iguais.

Comandos DDL

CREATE TABLE - Criação de tabela
ALTER TABLE - Adiciona, altera ou exclui tabelas
DROP TABLE - Comando para apagar a tabela

Comandos DML

INSERT - Usado para inserir dados na tabela
SELECT - Usado para visualizar a tabela

Comandos DQL

SELECT - Selecione um dado para visualização
JOIN - Serve para combinar tabelas

Existem diversas combinações de comandos para a visualização necessaria, no meu GitHub tenho diversos exemplos de exercicios usando operadores e parâmetros.

Normalização

A normalização é um processo de organização de tabelas em um banco de dados para reduzir redundâncias, evitar problemas de atualização e garantir a consistência dos dados, dividindo-os em formas normais. Isso ajuda a melhorar a eficiência e a integridade dos dados armazenados.

Bando de Dados NoSQL

Um banco de dados NoSQL é um sistema de banco de dados que não utiliza o modelo relacional. Ele é projetado para armazenar e processar dados não estruturados ou semiestruturados, oferecendo flexibilidade e escalabilidade horizontal, como por exemplo o MongoDB.

NoSQL oferece varias vantagens como permitir distribuir dados por múltiplos servidores, garantindo alta disponibilidade e expansão.
Permitem uma flexibilidade de esquema, um alto desempenho para operações rapidas de leitura e escrita, e uma facilidade em seu gerenciamento.

MongoDB

MongoDB é um banco de dados NoSQL orientado a documentos que armazena dados em formato JSON. Suas principais vantagens incluem escalabilidade horizontal e flexibilidade de esquema, ideal para aplicações que lidam com grandes volumes de dados variados.
